# インラインコメント挿入プロンプト
## 目的
指定のコードやMarkdown文書に対し、AIが**適切な位置にインラインコメントを自動挿入**するプロンプト。
このプロンプトをCursorに入力すると、対象コード全体に対して**処理意図を明確化するコメント**が自動で挿入される。

## 指示
1. コメント対象のコード・文書全体を入力として与える。  
2. 各行または各ブロックに対して、以下を明確にコメントとして付与する：
   - 処理の意図や目的  
   - 注意点・改善点  
   - 可読性向上のための補足  
3. コメントは対象言語の形式で記述（例：`//`、`#`、`<!-- -->`など）。  
4. コメントは**簡潔・具体的**にする（1行あたり1コメントが原則）。

## 制約
- コメントのみを追加し、元のコードロジックは変更しない。 
- コードの末尾にコメントしない。コードの上の行にコメントする。
- 不要な説明的コメント（例：「変数宣言です」など）は避ける。  
- コメントは**処理の意図**と**設計上の根拠**に重点を置く。  
- Markdown内では `<!-- コメント -->` 形式を使用する（[markdown_rule.md]に準拠）。  

## 期待する出力形式
```markdown
// コメント追加例
// エラー発生時に結果が未定義となる可能性あり
if (!result) {
　// エラーハンドリング関数を呼び出す
  handleError();
}
````

## 入力・出力例

### 入力
```javascript
function sum(a, b) {
  return a + b;
}
````

### 出力
```javascript
// 2つの数値を受け取り合計を返す
function sum(a, b) { 
  // シンプルな加算処理
  return a + b;      
}
```
