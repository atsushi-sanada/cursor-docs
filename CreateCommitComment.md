# Gitコミットメッセージ作成指示

## 指示内容
- Gitのコミットメッセージを作成するときは以下のルールに従って生成してください。

## 指示内容

- 可能な限り短く簡潔に書いてください
- 箇条書きにしてください
- タイトルと本文を分けてください。以下の推奨形式に従ってください。

## 推奨形式

```
<TITLE>
<空行>
<BODY>
```

### 具体例

```
TSVエクスポートとセル加工ロジックの追加。その他1件。

- TSVエクスポート機能を改善し、2行目のデータを追加。
- 空行をスキップする処理を実装し、セルの値を適切に加工するロジックを追加。
- 新しいsetup関数を作成し、初期設定完了時のアラート表示とボタン削除機能を実装。
```
